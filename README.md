# college-budget-web

대학생 생활비 관리와 소비 판단을 위한 React + Vite 프로젝트입니다.  
백엔드 없이 `localStorage`만 사용하고, 월 수입을 기준으로 하루 예산을 계산해 지출 가능 여부를 빠르게 확인하는 데 초점을 둡니다.

## 1. 프로젝트 개요

이 프로젝트의 핵심 목표는 다음 한 문장입니다.

`지금 이 소비를 해도 되는지`를 빠르게 판단한다.

기록 중심 가계부보다, 오늘 쓸 수 있는 금액을 보여주고 지출이 가능한지 바로 확인하는 흐름을 우선합니다.

## 2. 현재 구현 범위

- 월 수입 입력과 저장
- 하루 예산 계산
- 실시간 소비 가능 여부 판단
- 경고 메시지 표시
- 지출 기록 저장
- 카테고리별 기본 통계
- localStorage 기반 로그인 상태

## 3. 페이지 구성

- `#/` 메인 화면
- `#/budget-status` 예산 상태
- `#/expense-records` 지출 기록
- `#/statistics` 통계
- `#/login` 로그인
- `#/my-page` 마이페이지

## 4. localStorage 사용

이 프로젝트는 `src/lib/storage.js`에서만 localStorage를 다룹니다.

사용하는 키:

- `monthlyIncome`
- `budgetAmount`
- `expenseRecords`
- `loginState`

## 5. 핵심 계산 로직

### 5.1 하루 예산 계산

- 하루 예산 = 월 수입 / 이번 달 남은 일수
- 월 수입이 비어 있거나 0 이하이면 하루 예산은 0으로 처리합니다.

### 5.2 소비 판단

- 입력 지출 금액이 하루 예산 이하이면 사용 가능
- 입력 지출 금액이 하루 예산 초과이면 경고

### 5.3 입력 예외 처리

- 빈값
- 숫자가 아닌 입력
- 0 이하 입력

## 6. 지출 기록

- 지출 금액, 카테고리, 날짜, 메모를 저장합니다.
- 지출 금액은 숫자만 허용합니다.
- 저장된 지출은 바로 목록과 통계에 반영됩니다.

## 7. 통계

- 저장된 지출 총액
- 기록 개수
- 카테고리별 지출 비중

## 8. 로그인

- 실제 인증이 아닌 UI 상태 저장입니다.
- 이름을 입력하면 로그인 상태가 저장됩니다.
- 마이페이지에서 현재 상태를 확인하고 로그아웃할 수 있습니다.

## 9. 실행 방법

```bash
npm install
npm run dev
```

빌드 확인:

```bash
npm run build
```
