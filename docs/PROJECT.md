# PROJECT.md

## 1. 프로젝트 개요

이 프로젝트는 대학생을 위한 생활지출 관리 웹 서비스이다.

핵심 목표:
👉 “지금 이 소비를 해도 되는지”를 실시간으로 판단

기존 가계부와의 차이:
- 소비 기록 중심 ❌
- 소비 판단 중심 ⭕

## 4. 데이터 처리 방식

- 기본: 더미 데이터
- 상태 유지: localStorage 사용 가능

### 규칙

- localStorage 접근은 lib에서 관리
- 키는 명확하게 정의

## 5. 로그인 기능 정의

로그인 기능은 구현 가능하다.

### 허용

- 로그인 UI
- 더미 사용자 데이터
- localStorage 기반 로그인 상태
- 로그인/로그아웃 상태 관리

### 금지

- 서버 인증
- JWT / OAuth
- API 호출

👉 로그인은 “UI 상태”이며 인증 시스템이 아니다.

---

## 6. 핵심 기능 범위

### 포함

- 월 수입 입력
- 하루 예산 계산
- 지출 입력
- 소비 가능 여부 판단
- 경고 메시지
- 지출 기록
- 통계 (차트)

### 제외

- 금융 API
- 투자 기능
- AI 추천

---

## 7. 핵심 로직

- 하루 예산 = 남은 금액 / 남은 일수

### 소비 판단

- 예산 이하 → 가능
- 초과 → 경고

### 예외 처리

- 빈 값
- 음수
- NaN
- 남은 일수 0
